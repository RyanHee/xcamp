# Description:

Any natural number n greater than 1 can be divided into the sum of several natural numbers less than n. When n=7, there are 14 splitting methods:

7=1+1+1+1+1+1+1<br>
7=1+1+1+1+1+2<br>
7=1+1+1+1+3<br>
7=1+1+1+2+2<br>
7=1+1+1+4<br>
7=1+1+2+3<br>
7=1+1+5<br>
7=1+2+2+2<br>
7=1+2+4<br>
7=1+3+3<br>
7=1+6<br>
7=2+2+3<br>
7=2+5<br>
7=3+4<br>
 

# Input:

A number n（2≤n≤25）

 

# Output:

Detailed splitting method

 

# Sample input：

5
# Sample output：

5=1+1+1+1+1<br>
5=1+1+1+2<br>
5=1+1+3<br>
5=1+2+2<br>
