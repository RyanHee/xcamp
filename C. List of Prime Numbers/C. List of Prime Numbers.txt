# Time Limit: 1s Memory Limit: 256M
 
# Description
A prime number is a number that can only be divided by 1 and itself. For example: 2, 3, 5, 7, 11, 13, and 17 are prime numbers. Given an integer n, please find all the prime numbers between 1 and n.

 

# Input
An integer n. (1≤n≤2000)

# Output
All prime numbers that are less than n in ascending order, separated by spaces.

 

# Sample Input
20

# Sample Output
2 3 5 7 11 13 17 19
